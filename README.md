# 📝 ToDo App (Fullstack Projekt)

## 🚀 Projektbeschreibung
Diese ToDo-App ist ein Fullstack-Übungsprojekt, das ich entwickelt habe, um moderne Webentwicklung zu lernen.  
Die Anwendung ermöglicht das Erstellen, Löschen und Verwalten von Aufgaben mit einer eigenen API und Datenbank.

---

## 🧠 Features
- Aufgaben erstellen
- Aufgaben löschen
- Aufgaben als erledigt markieren
- Speicherung in SQLite Datenbank
- REST API mit Flask
- Frontend mit HTML, CSS und JavaScript

---

## 🛠️ Tech Stack
**Frontend:**
- HTML
- CSS
- JavaScript (Vanilla)

**Backend:**
- Python
- Flask
- Flask-CORS

**Datenbank:**
- SQLite
